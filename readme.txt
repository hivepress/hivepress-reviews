== Description ==

HivePress Reviews is an extension for [HivePress](https://wordpress.org/plugins/hivepress/) plugin. It allows users to rate and review listings.

[Demo](https://demo.hivepress.io/) | [Docs](https://hivepress.io/docs/extensions/reviews/) | [Support](https://hivepress.io/support/forum/extensions/reviews/)
