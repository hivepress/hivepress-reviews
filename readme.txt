== Description ==

HivePress Reviews is an extension for [HivePress](https://wordpress.org/plugins/hivepress/) plugin. It allows users to rate and review listings.

[Demo](https://listinghive.hivepress.io/) | [Docs](https://help.hivepress.io/category/40-reviews) | [Support](https://community.hivepress.io/?utm_medium=referral&utm_source=wordpress.org)
