== Description ==

HivePress Reviews is an extension for [HivePress](https://wordpress.org/plugins/hivepress/) plugin. It allows users to rate and review listings.

[Demo](https://demo.hivepress.io/) | [Docs](https://hivepress.io/docs/extensions/reviews/?utm_medium=referral&utm_source=wordpress.org) | [Support](https://hivepress.io/support/forum/extensions/reviews/?utm_medium=referral&utm_source=wordpress.org)
