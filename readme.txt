Reviews extension for HivePress plugin.

== Description ==

HivePress Reviews is an extension for [HivePress](https://wordpress.org/plugins/hivepress/) plugin. It allows users to rate and review listings.

[Demo](https://demo.hivepress.co/) | [Documentation](https://docs.hivepress.co/) | [Support](https://github.com/hivepress/hivepress/issues)

== Installation ==

= Automatic installation =

To do an automatic install of HivePress Reviews, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “HivePress Reviews” and click Search Plugins. Once you’ve found the plugin you can install it by simply clicking “Install Now”.

= Manual installation =

Download the plugin, log in to your WordPress dashboard and navigate to the Plugins menu, then click Add New.

Click Upload and select the downloaded ZIP file, and finally click “Install Now”.
